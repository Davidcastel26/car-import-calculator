# auto-market

A transparent calculator for vehicle import costs in Guatemala. Eliminates the "black box" of IPR, DAI, and IVA calculations.

## Features

- IPR (Impuesto a la Primera Matrícula) calculation
- DAI (Derechos Arancelarios a la Importación) estimation
- IVA (12%) computation
- Gestoría/Tramitador fee validation
- Open-source, licensed under Apache 2.0
